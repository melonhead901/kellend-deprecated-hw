UW CSE 378 was a class covering Machine Organization and Assmebly Language.

I took this class in Fall 2010, it was taught by Mark Oskin.

Course website is here: http://www.cs.washington.edu/education/courses/cse378/10au/

Brief homework descriptions:
HW 1 - Annotating compiled x86 source code
HW 2 - Implementing str str in x86-64
HW 3 - Implementing quicksort in x86-64
HW 4 - Understanding a variety of cache geometries
